Small unformatted APS-files with dummy background concentrations. 
In order to test whether all files are present and have the correct name.
Note that the file header is in all files the same and not a correct file header.

Chem_meteo_actual/   : dummy background files, actual meteo; names as in actual data-directory
Chem_meteo_prognosis/: dummy background files, prognosis meteo; names as in actual data-directory
Chem_meteo_template/ : small template files; also script to copy all files to Chem_meteo_actual and Chem_meteo_prognosis.
                       template3x2.ops     : 3x2 grid,  1 field
                       template3x2_nf3.ops : 3x2 grid,  3 fields
                       template3x2_nf12.ops: 3x2 grid, 12 fields
